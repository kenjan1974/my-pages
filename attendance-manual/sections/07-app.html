行動 APP

行動 APP 供全體員工在手機上使用，可進行上下班打卡、提送請假 / 加班 / 補卡申請、查看出勤紀錄與通知。APP 支援 iOS 與 Android 雙平台。

APP 底部共有 5 個主要頁籤：

頁籤 圖示 說明
打卡 🏠 上下班打卡（首頁），顯示今日打卡狀態與 GPS / Wi-Fi 驗證結果
申請單 📋 提送與查看請假、加班、補卡申請
出勤紀錄 📅 以月曆檢視每日打卡紀錄與異常標記
通知 🔔 接收簽核結果、異常提醒等系統通知
個人 👤 個人資訊、LINE 綁定、密碼變更等
APP 主畫面（打卡頁籤）— 顯示五個底部頁籤的全貌

登入與 LINE 綁定

帳號密碼登入

  1. 開啟 APP，進入登入畫面
  2. 輸入員工編號（例如 EMP001）
  3. 輸入密碼（預設為管理員配發的初始密碼）
  4. 點擊「登入」按鈕
  5. 首次登入成功後會自動進入打卡首頁
APP 登入畫面 — 員工編號與密碼欄位、登入按鈕、LINE 登入按鈕

LINE 快速登入

若已完成 LINE 帳號綁定，可直接點擊登入畫面的「使用 LINE 登入」按鈕，透過 LINE OAuth 驗證後自動登入。

LINE 帳號綁定

  1. 先以帳號密碼登入 APP
  2. 前往「個人」頁籤 → 點擊「LINE 帳號綁定」
  3. APP 會導向 LINE 授權頁面，點擊「允許」
  4. 授權成功後返回 APP，顯示「LINE 已綁定」
  5. 之後即可在登入畫面使用「使用 LINE 登入」快速登入
LINE 綁定畫面 — 顯示綁定前的引導頁面
LINE 綁定成功 — 顯示已綁定狀態
⚠️ 注意 每個 LINE 帳號只能綁定一位員工。若需解除綁定，請聯繫管理員。

打卡

打卡頁面是 APP 的首頁，進入後系統會自動偵測目前的位置與 Wi-Fi 狀態，判斷是否在公司範圍內。

打卡驗證方式

系統同時採用 GPS 定位與Wi-Fi SSID 雙重驗證：

  • GPS 驗證：根據手機定位座標，計算與公司位置的距離，在設定的圍欄半徑範圍內視為「在公司」
  • Wi-Fi 驗證：偵測目前連線的 Wi-Fi 名稱，與公司設定的 SSID 比對。若公司啟用「Wi-Fi 必要模式」，則必須連上公司 Wi-Fi 才能正常打卡

正常打卡（公司範圍內）

  1. 開啟 APP，自動進入打卡頁面
  2. 頁面上方顯示目前 GPS 狀態與 Wi-Fi 連線狀態
  3. 當顯示「在公司範圍內」時，按下「上班打卡」按鈕
  4. 系統跳出確認對話框，點擊「確認打卡」
  5. 打卡成功後頁面會更新顯示打卡時間
  6. 下班時按下「下班打卡」按鈕，流程相同
打卡頁面 — 在公司範圍內，顯示「上班打卡」按鈕與位置狀態
打卡確認對話框 — 「確認上班打卡」提示
打卡成功 — 頁面顯示已打上班卡的時間

外勤打卡（不在公司範圍）

若員工因外出洽公等原因不在公司範圍內，系統會提示進行外勤打卡：

  1. 頁面顯示「不在公司範圍內」，並標示與公司的距離
  2. 按下打卡按鈕後，系統跳出外勤打卡確認對話框
  3. 確認後系統會開啟相機要求拍照，作為外勤打卡佐證
  4. 拍照完成後送出打卡，記錄會標記為「外勤」
外勤打卡提示 — 顯示不在範圍內與距離資訊
外勤打卡拍照 — 開啟相機拍照作為佐證
💡 提示
  • 若 Wi-Fi 必要模式啟用但手機未開啟 Wi-Fi，系統會提示開啟 Wi-Fi 設定
  • 若 GPS 權限被拒絕，系統會引導開啟定位權限
  • 上班打卡超過 09:35:59 即記為「遲到」；下班打卡早於應下班時間即記為「早退」
打卡頁面 — 已完成上下班打卡，顯示兩次打卡時間

申請單

點擊底部「申請單」頁籤，進入申請單管理頁面。頂部有三個子頁籤：請假、加班、補卡，可切換檢視不同類型的申請。

申請單列表頁 — 顯示三個子頁籤（請假 / 加班 / 補卡）與申請紀錄列表

請假申請

  1. 在「請假」子頁籤下，點擊右下角的「+」浮動按鈕
  2. 進入請假申請表單：
    • 選擇假別（特休、事假、病假等，僅顯示有額度的假別）
    • 選擇的假別下方會顯示可用餘額
    • 選擇起始日期 / 時間和結束日期 / 時間
    • 系統自動計算請假時數
    • 填寫請假事由
  3. 確認無誤後點擊「送出」按鈕
  4. 申請送出後狀態為「待審核」，等待主管簽核
請假申請表單 — 假別選擇、日期時間選擇、餘額顯示、事由欄位
請假申請詳情 — 顯示已送出申請的完整資訊與審核狀態

加班申請

  1. 切換到「加班」子頁籤，點擊「+」浮動按鈕
  2. 填寫加班申請表單：
    • 選擇加班日期
    • 選擇起始時間和結束時間
    • 系統自動計算加班時數
    • 填寫加班事由
  3. 確認後點擊「送出」
加班申請表單 — 日期、時間區間、事由欄位

補卡申請

  1. 切換到「補卡」子頁籤，點擊「+」浮動按鈕
  2. 填寫補卡申請表單：
    • 選擇補卡日期（限事發後 3 個日曆天內）
    • 選擇補卡類型（上班 / 下班）
    • 選擇補卡時段（上班卡：08:30 / 14:00；下班卡：12:30 / 18:00）
    • 填寫補卡原因
  3. 確認後點擊「送出」
補卡申請表單 — 日期、補卡類型、固定時段選擇、原因欄位
⚠️ 補卡限制
  • 僅能補卡3 個日曆天內的紀錄
  • 同一天最多2 筆補卡申請
  • 已有 2 筆打卡紀錄的日期不可再補卡

申請狀態查看

每筆申請會以卡片方式呈現，顯示申請類型、日期、狀態標籤：

  • 待審核 — 等待主管處理
  • 已核准 — 主管已批准
  • 已駁回 — 主管已退回，可查看駁回原因
  • 已撤銷 — 員工自行撤銷

點擊任一申請卡片可進入詳情頁面，查看完整申請資訊與簽核結果。

出勤紀錄

點擊底部「出勤紀錄」頁籤，以月曆檢視方式查看整月的出勤狀況。

月曆畫面

  • 每個日期格子會以顏色標記標示當日出勤狀態（正常、異常、請假、缺勤等）
  • 點擊任一日期可快速展開該日的打卡紀錄詳情：上班時間、下班時間、打卡方式、是否異常
  • 如該日有異常（遲到、早退、缺勤），可直接從月曆點擊快速補卡
  • 使用左右箭頭或滑動切換月份
出勤紀錄月曆 — 整月出勤狀態總覽，以顏色標記正常 / 異常 / 請假 / 缺勤
出勤紀錄日詳情 — 點擊日期後展開的打卡時間與狀態資訊

月報表

在出勤紀錄頁面右上角點擊「月報表」按鈕，進入個人月報表畫面。

報表內容

  • 統計磚塊：出勤天數、遲到次數、早退次數、請假天數、加班時數等快速總覽
  • 每日紀錄清單：依日期排列的詳細打卡紀錄，包含上下班時間、打卡方式、工時與異常標記
  • 可切換月份查看不同月份的統計資料
月報表統計磚塊 — 出勤天數、遲到 / 早退次數、請假天數等統計數字
月報表每日紀錄 — 依日期排列的打卡紀錄詳情

通知中心

點擊底部「通知」頁籤，查看所有系統通知。底部頁籤的通知圖示會顯示未讀數量紅點。

通知類型

  • 簽核結果：申請被核准或駁回時收到通知
  • 異常提醒：打卡異常（遲到、早退、缺勤）的提醒
  • 系統公告：公司管理員發送的公告訊息

操作方式

  • 未讀通知以醒目樣式顯示，已讀則為較淡的樣式
  • 點擊通知可標示為已讀並檢視詳細內容
  • 簽核結果類通知點擊後可直接跳轉到對應的申請詳情頁面
通知列表 — 顯示已讀與未讀通知，含簽核結果、異常提醒等
通知頁籤紅點 — 底部導覽列通知圖示上的未讀數量提示

個人資料

點擊底部「個人」頁籤，查看與管理個人帳號資訊。

頁面內容

  • 個人資訊卡片：顯示姓名、員工編號、部門、職稱與角色標籤
  • LINE 綁定狀態：顯示目前 LINE 帳號是否已綁定，未綁定可點擊進行綁定
  • 快捷功能：變更密碼、LINE 帳號綁定 / 解除、關於系統（版本資訊）
  • 登出：點擊最下方「登出」按鈕，清除登入狀態並返回登入頁面
個人資料頁面 — 個人資訊卡片、LINE 綁定狀態、快捷功能選項
變更密碼畫面 — 輸入目前密碼與新密碼
