自助服務

管理員 老闆 一般員工 — 所有已登入的使用者皆可使用自助服務功能。

我的請假

此頁面顯示您個人的所有請假申請紀錄，並可新增請假申請或撤銷待審核的申請。

檢視請假紀錄

  • 進入後預設顯示所有請假紀錄，可依狀態篩選（待審核、已核准、已駁回）
  • 每筆紀錄顯示：假別、起始日、結束日、時數、單位（整天/半天/小時）、事由、狀態、申請時間
  • 使用表格底部的分頁功能瀏覽更多紀錄
我的請假申請 — 紀錄列表，可篩選狀態並撤銷待審核的申請

新增請假申請

  1. 點擊右上角「＋ 新增請假」按鈕
  2. 選擇假別（如事假、特休、病假、公假等）
  3. 依假別自動顯示可用的請假單位（整天、半天、小時），選擇對應單位
  4. 選擇日期範圍（起始日～結束日）；依單位不同，可能需要選擇上午/下午或指定時段
  5. 填寫事由（選填，但建議填寫以利簽核）
  6. 點擊「送出申請」
新增請假申請表單 — 選擇假別後會自動帶出可用的請假單位
💡 提示 送出後，系統會自動判斷簽核層級並派送給對應的簽核人。您可以在紀錄列表中追蹤申請狀態。

撤銷請假申請

  • 僅限「待審核」狀態的申請才可撤銷
  • 在紀錄列表中，點擊對應項目的「撤銷」按鈕
  • 確認後申請狀態會變更為「已撤銷」

我的加班

此頁面顯示您個人的所有加班申請紀錄，並可新增加班申請。

檢視加班紀錄

  • 預設顯示所有加班申請，可依狀態篩選
  • 每筆紀錄顯示：加班類型（平日/假日）、日期、時段、時數、原因、狀態
我的加班申請 — 加班紀錄列表

新增加班申請

  1. 點擊右上角「＋ 新增加班」按鈕
  2. 選擇加班類型（平日加班 / 假日加班）
  3. 選擇加班日期
  4. 設定起始時間與結束時間，系統會自動計算加班時數
  5. 填寫加班原因
  6. 點擊「送出申請」
新增加班申請表單 — 選擇日期與時段後自動計算時數

我的補卡

如果您忘記打上班卡或下班卡，可透過此頁面申請補卡。

檢視補卡紀錄

  • 顯示所有補卡申請紀錄，可依狀態篩選
  • 每筆紀錄顯示：補卡類型（上班卡 / 下班卡）、日期、補卡時間、原因、狀態
我的補卡申請 — 補卡紀錄列表

新增補卡申請

  1. 點擊右上角「＋ 新增補卡」按鈕
  2. 選擇補卡類型（補上班卡 / 補下班卡）
  3. 選擇補卡日期（僅可選擇 3 個日曆天以內的日期）
  4. 選擇補卡時段（固定選項：上班卡 08:30 / 14:00；下班卡 12:30 / 18:00）
  5. 填寫補卡原因
  6. 點擊「送出申請」
新增補卡申請表單 — 選擇日期與固定補卡時段
⚠️ 補卡限制
  • 須在事發後 3 個日曆天內提出申請，逾期無法申請
  • 當天打卡紀錄已有 2 筆（含）以上時，不得申請補卡
  • 每天最多申請 2 筆補卡（上班卡與下班卡需分開申請）
  • 補卡申請須經主管核准後才會計入出勤紀錄

假期餘額

查看您個人在各假別的剩餘額度。

  • 頁面顯示當年度各假別的總額度、已使用、剩餘，並以進度條方式直覺呈現
  • 可切換年度查看歷年紀錄
  • 僅顯示有額度的假別（如特休、事假、病假等）
假期餘額 — 以進度條顯示各假別使用情況

我的出勤

查看每日打卡、請假、加班與補卡關聯資訊。

頁面結構

  • 頂端顯示員工基本資訊：姓名、工號、部門、班別
  • 左上方「今日狀態」卡片：顯示今日日期、出勤狀態 Tag、上班時間、下班時間，以及狀態提示文字
  • 右上方顯示當月 6 項統計：出勤天數、遲到次數、早退次數、請假時數、加班時數、外勤天數
  • 右上角提供「查看月報表」與「重新整理」按鈕

月曆與單日明細

  • 月曆以色點標示每日事件，圖例說明（依序）：
    • ● 正常（綠）／● 遲到（橘黃）／● 早退（橘）／● 缺卡・缺席（紅）
    • ● 外勤（紫）／○ 請假（藍框空心）／● 加班（青）／● 補卡（深藍）
  • 使用「上月」「下月」按鈕切換月份（當月不可跳至未來）
  • 點擊月曆日期後，右側「單日出勤與申請明細」面板即時更新，顯示：
    • 「當日打卡摘要」：上班時間（含打卡方式：GPS / Wi-Fi / 外勤）、下班時間、工時（累計）、遲到 / 早退分鐘數
    • 「當日事件」：該日的請假、加班、補卡申請清單（含類型、時數與狀態）
    • 「當日摘要」：合計請假 N 筆、加班 N 筆、補卡 N 筆
我的出勤 — 月曆（左）與單日明細面板（右），點選日期即時顯示打卡摘要與當日事件
ℹ️ 免打卡員工注意 老闆與設定為「免打卡」的員工不顯示出勤統計、月曆及單日明細，頁面改為顯示提示訊息，並提供前往我的請假、我的加班、我的補卡的快捷按鈕。

月出勤報表

在「我的出勤」頁面點擊右上角「查看月報表」，即可進入個人月出勤報表頁面。

  • 頂端同樣顯示 6 項統計卡片（出勤天數、遲到次數、早退次數、請假時數、加班時數、外勤天數）
  • 本月摘要卡片：已記錄天數、異常天數、最近一筆紀錄日期
  • 本月提醒卡片：依實際資料顯示提示，例如「本月目前有 N 天出現遲到、早退或缺卡狀態」或「本月目前出勤狀態穩定」；另列出當月請假時數、加班時數、外勤天數提示
  • 每日出勤明細表格欄位：日期、星期、上班、下班、狀態、遲到（分）、早退（分）、請假、加班、外勤、備註
  • 備註欄自動顯示：尚未完成下班打卡 / 補卡 N 筆 / 工時 Xh
  • 使用「上月」「下月」切換月份；點擊右上角「回到出勤紀錄」可返回月曆頁面
月出勤報表 — 當月統計卡片、本月摘要 / 提醒，以及每日出勤明細表格

快捷功能

個人資料下方的「快捷功能」卡片提供兩個快速跳轉按鈕：

  • 「我的出勤」— 跳轉至月曆打卡頁面
  • 「月出勤報表」— 跳轉至個人月出勤報表

變更密碼

  1. 點擊頁面底部「變更密碼」按鈕
  2. 在彈出視窗中依序填寫：目前密碼、新密碼、確認新密碼
  3. 新密碼規則：至少 8 個字元，且需同時包含英文與數字
  4. 點擊「確認變更」送出；成功後視窗自動關閉
💡 提示 如需修改部門、班別或角色等資訊，請聯繫管理員在「員工管理」中調整。「關於系統」按鈕目前尚未開放。

我的簽核

當您擔任主管或被指定為代理簽核人時，可在此頁面處理下屬或指定人員的申請單。

待簽核

  • 頁籤標題顯示目前待處理數量，例如「待簽核 (3)」
  • 每筆紀錄顯示：類型（請假 / 加班 / 補卡）、申請人、步驟（「第 N 步」）、摘要、申請時間
  • 「操作」欄直接顯示「核准」與「駁回」兩個按鈕
  • 點擊任一按鈕後，系統會彈出確認視窗，顯示申請詳情，並提供備註（選填）欄位，確認後點「確認核准」或「確認駁回」完成操作
  • 無待簽核申請時顯示「目前沒有待簽核的申請」
我的簽核 — 待簽核頁籤，每筆申請列有「核准」與「駁回」按鈕

已簽核

  • 顯示您已處理過（核准或駁回）的歷史簽核紀錄
  • 篩選器：年份（當年及前三年）、表單類型（請假 / 加班 / 補卡）、申請人（以「工號 姓名」格式列出）
  • 每筆紀錄顯示：類型、申請人、摘要、申請時間、簽核時間、結果（核准 / 駁回）
  • 清單支援分頁（每頁 20 筆）；無資料時顯示「目前沒有已簽核的紀錄」
我的簽核 — 已簽核頁籤，可依年份、類型與申請人篩選歷史紀錄
💡 提示 「我的簽核」與管理員使用的「簽核管理 → 簽核中心」不同：簽核中心供管理員統一查看全公司所有申請；「我的簽核」僅顯示指派給您本人的簽核任務。
