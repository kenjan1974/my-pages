系統簡介

差勤管理系統是專為中小企業設計的出勤管理平台，涵蓋員工打卡、請假、加班、補卡等日常差勤流程，並提供主管簽核、異常偵測、報表產出等管理功能。

系統由兩個主要介面組成：

  • 行動 APP（員工端）：員工透過手機進行上下班打卡、查看出勤紀錄、提送各類申請
  • 管理後台（Web）：管理人員透過電腦瀏覽器進行人事管理、簽核處理、報表查詢與匯出

本手冊為管理後台的操作說明文件。

💡 系統特色 支援 LINE 快速登入與推播通知、Wi-Fi + GPS 雙重打卡驗證、自動異常偵測引擎、兩階段簽核流程、Excel 報表匯出。

角色與權限

系統定義三種角色，各角色可使用的功能範圍不同：

角色 說明 後台可用功能
管理員 人事管理人員，負責系統設定與人事作業 全部功能（自助服務 + 簽核管理 + 報表中心 + 基礎資料管理）
老闆 公司負責人，為最終簽核人 全部功能（同管理員）
一般員工 一般員工；若被指派為部門主管，可簽核部屬的申請 僅限「自助服務」（我的請假、我的加班、我的補卡、假期餘額）
ℹ️ 部門主管 員工若被設定為某部門的主管（在「部門管理」中指定），該員工會成為部屬申請的第一階簽核人。部門主管本身仍是「員工」角色，但具備簽核權限。

登入系統

開啟瀏覽器，進入管理後台網址。系統會顯示登入頁面。

帳號密碼登入

  1. 在登入頁面輸入員工編號（如 ADMIN001）
  2. 輸入密碼
  3. 點擊「登入」按鈕
  4. 登入成功後，管理員／老闆會導向「簽核中心」；一般員工會導向「我的請假」
登入頁面截圖
⚠️ 注意 連續多次輸入錯誤密碼可能會暫時鎖定帳號。請確認員工編號與密碼無誤後再嘗試。

介面導覽

登入後，管理後台的介面由以下區域組成：

  • 頂部標題列：顯示系統名稱「差勤管理後台」及目前登入的使用者名稱與角色
  • 左側導覽選單：依角色顯示可用功能項目，分為「自助服務」「簽核管理」「報表中心」「基礎資料」四大群組
  • 主要內容區：點擊左側選單項目後，右側會顯示對應的頁面內容
管理後台介面總覽 — 左側為功能選單，右側為內容區域（此為管理員視角）

一般員工登入後，左側選單僅顯示「自助服務」群組（我的請假、我的加班、我的補卡、假期餘額）。

自助服務

管理員 老闆 一般員工 — 所有已登入的使用者皆可使用自助服務功能。

我的請假

此頁面顯示您個人的所有請假申請紀錄，並可新增請假申請或撤銷待審核的申請。

檢視請假紀錄

  • 進入後預設顯示所有請假紀錄，可依狀態篩選（待審核、已核准、已駁回）
  • 每筆紀錄顯示：假別、起始日、結束日、時數、單位（整天/半天/小時）、事由、狀態、申請時間
  • 使用表格底部的分頁功能瀏覽更多紀錄
我的請假申請 — 紀錄列表，可篩選狀態並撤銷待審核的申請

新增請假申請

  1. 點擊右上角「＋ 新增請假」按鈕
  2. 選擇假別（如事假、特休、病假、公假等）
  3. 依假別自動顯示可用的請假單位（整天、半天、小時），選擇對應單位
  4. 選擇日期範圍（起始日～結束日）；依單位不同，可能需要選擇上午/下午或指定時段
  5. 填寫事由（選填，但建議填寫以利簽核）
  6. 點擊「送出申請」
新增請假申請表單 — 選擇假別後會自動帶出可用的請假單位
💡 提示 送出後，系統會自動判斷簽核層級並派送給對應的簽核人。您可以在紀錄列表中追蹤申請狀態。

撤銷請假申請

  • 僅限「待審核」狀態的申請才可撤銷
  • 在紀錄列表中，點擊對應項目的「撤銷」按鈕
  • 確認後申請狀態會變更為「已撤銷」

我的加班

此頁面顯示您個人的所有加班申請紀錄，並可新增加班申請。

檢視加班紀錄

  • 預設顯示所有加班申請，可依狀態篩選
  • 每筆紀錄顯示：加班類型（平日/假日）、日期、時段、時數、原因、狀態
我的加班申請 — 加班紀錄列表

新增加班申請

  1. 點擊右上角「＋ 新增加班」按鈕
  2. 選擇加班類型（平日加班 / 假日加班）
  3. 選擇加班日期
  4. 設定起始時間與結束時間，系統會自動計算加班時數
  5. 填寫加班原因
  6. 點擊「送出申請」
新增加班申請表單 — 選擇日期與時段後自動計算時數

我的補卡

如果您忘記打上班卡或下班卡，可透過此頁面申請補卡。

檢視補卡紀錄

  • 顯示所有補卡申請紀錄，可依狀態篩選
  • 每筆紀錄顯示：補卡類型（上班卡 / 下班卡）、日期、補卡時間、原因、狀態
我的補卡申請 — 補卡紀錄列表

新增補卡申請

  1. 點擊右上角「＋ 新增補卡」按鈕
  2. 選擇補卡類型（補上班卡 / 補下班卡）
  3. 選擇補卡日期（僅可選擇 3 個日曆天以內的日期）
  4. 設定補卡時間（HH:MM 格式，可選 00:00～23:59）
  5. 填寫補卡原因
  6. 點擊「送出申請」
新增補卡申請表單 — 選擇日期與時間
⚠️ 補卡限制
  • 須在事發後 3 個日曆天內提出申請，逾期無法申請
  • 當天打卡紀錄已有 2 筆（含）以上時，不得申請補卡
  • 每天最多申請 2 筆補卡（上班卡與下班卡需分開申請）
  • 補卡申請須經主管核准後才會計入出勤紀錄

假期餘額

查看您個人在各假別的剩餘額度。

  • 頁面顯示當年度各假別的總額度、已使用、剩餘，並以進度條方式直覺呈現
  • 可切換年度查看歷年紀錄
  • 僅顯示有額度的假別（如特休、事假、病假等）
假期餘額 — 以進度條顯示各假別使用情況

簽核管理

管理員 老闆 — 僅管理員與老闆可使用簽核管理功能。

簽核中心

簽核中心是處理員工請假、加班、補卡申請的統一入口。管理員與老闆登入後，預設會導向此頁面。

待簽核列表

  • 頁面分為「待簽核」和「已簽核」兩個 Tab 頁籤
  • 待簽核列表顯示所有等待您核准的申請：類型（請假/加班/補卡）、申請人、簽核步驟、摘要、申請時間
  • 每筆申請旁有「核准」與「駁回」兩個操作按鈕
簽核中心 — 待簽核列表，可直接進行核准或駁回操作

核准 / 駁回申請

  1. 在待簽核列表中，找到要處理的申請
  2. 點擊「✓ 核准」或「✕ 駁回」按鈕
  3. 若駁回，可填寫備註說明駁回原因
  4. 確認後申請狀態會更新，並通知申請人
💡 簽核流程說明 系統採兩階段簽核機制。一般員工的申請先由部門主管（第一關）核准，若假別設定為需兩關簽核，則還須由老闆（第二關）做最終核准。部門主管自己的申請則直接由老闆簽核。

已簽核紀錄

切換到「已簽核」Tab 可查看您過去核准/駁回的歷史紀錄。

請假紀錄

查看全公司所有員工的請假申請紀錄。

  • 可依狀態篩選（待審核、已核准、已駁回、已撤銷）
  • 顯示完整資訊：申請人、假別、起始日、結束日、時數、事由、簽核狀態、簽核備註等
  • 支援分頁瀏覽
請假紀錄 — 全公司請假申請紀錄總覽

加班紀錄

查看全公司所有員工的加班申請紀錄。

  • 可依狀態篩選
  • 顯示：申請人、加班類型（平日/假日）、日期、時段、時數、原因、狀態等
加班紀錄 — 全公司加班申請紀錄總覽

補卡紀錄

查看全公司所有員工的補卡申請紀錄。

  • 可依狀態篩選
  • 顯示：申請人、補卡類型（上班卡/下班卡）、日期、補卡時間、原因、狀態等
補卡紀錄 — 全公司補卡申請紀錄總覽

報表中心

管理員 老闆 — 報表中心提供各種出勤統計與異常管理功能。

每日出勤總覽

以日期為維度，查看全公司當日的出勤狀況一覽。

頁面說明

  • 右上角可選擇日期，切換不同日期的出勤資料
  • 摘要卡片：顯示當日總人數、正常出勤、遲到、請假、曠職/缺卡、早退等各項統計
  • 員工出勤表格：列出每位員工的員工編號、姓名、部門、班別、上班打卡、下班打卡、打卡方式、外勤、遲到分鐘、狀態等
每日出勤總覽 — 頂部為統計摘要卡片，下方為全員打卡狀態明細

月出勤報表

以月份為維度，彙總全公司每位員工的出勤統計。

頁面說明

  • 可選擇年份、月份和部門篩選
  • 頂部顯示：該月應出勤天數、員工人數
  • 表格欄位包含：員工編號、姓名、部門、應出勤天、實出勤天、缺勤天、遲到、早退、外勤天、加班時數、誤餐費、請假時數
  • 右上角可點擊「匯出 Excel」按鈕下載報表檔案
月出勤報表 — 可依月份、部門篩選並匯出 Excel

薪資輔助報表

專為薪資計算提供的出勤資料報表，可直接作為薪資計算的輸入依據。

頁面說明

  • 可選擇年月與部門篩選
  • 報表包含薪資計算所需的關鍵欄位：出勤天數、遲到基數、早退基數、平日加班時數、假日加班時數、誤餐費、各假別請假天數/時數等
  • 右上角可點擊「匯出 Excel」下載，直接用於薪資作業
薪資輔助報表 — 包含遲到基數、加班分類等薪資計算所需資料

異常管理

異常管理儀表板用於集中查看並處理系統自動偵測到的出勤異常紀錄。

頁面說明

  • 摘要卡片：顯示待處理、已處理、遲到、早退等異常數量統計
  • 篩選條件：可依日期區間、異常類型、處理狀態進行篩選
  • 異常列表：顯示日期、員工編號、姓名、異常類型、嚴重程度、說明、狀態
異常管理 — 頂部為統計摘要，可篩選異常類型與狀態

處理異常紀錄

對於每筆異常紀錄，管理員可進行以下操作：

  • 標記「已確認」：確認此異常為正常情況（如主管口頭允許遲到）
  • 標記「免計」：免除扣除（如公務外出造成的異常）
  • 核准補卡：針對缺卡類異常，可直接核准對應的補卡申請
💡 異常類型說明 系統自動偵測的異常類型包含：遲到、早退、缺上班卡、缺下班卡、曠職、GPS 異常、Wi-Fi 異常、工時不足等。每種異常會依嚴重程度標記為「警告」或「嚴重」。

異常統計

以圖表方式呈現異常紀錄的統計分析，協助管理者掌握出勤異常趨勢。

  • 可選擇日期區間進行查詢
  • 異常類型長條圖：各類異常的數量分佈
  • 嚴重程度圓餅圖：警告與嚴重異常的比例
  • 明細卡片：各部門、各員工的異常次數排行
異常統計 — 以長條圖與圓餅圖呈現異常趨勢

年度統計

以年度為維度，呈現全公司的出勤與假別使用統計。

  • 特休使用率：全公司及各部門的特休使用率統計
  • 各部門出勤率：以長條圖/圓餅圖呈現各部門的年度出勤表現
  • 假別統計：各假別的年度使用量統計
  • 個人明細：可查看各員工的年度出勤詳情
年度統計 — 特休使用率、部門出勤率等年度統計圖表

自訂區間報表

可自由選擇任意日期範圍產出出勤報表，不限於月份維度。

  • 使用日期範圍選擇器指定起迄日期
  • 可依部門篩選
  • 報表欄位與月出勤報表相同：出勤天數、遲到、早退、加班、請假等
自訂區間報表 — 自由選擇日期範圍產出報表

基礎資料管理

管理員 老闆 — 基礎資料管理涵蓋公司核心設定，建議由管理員（人事）進行日常維護。

公司設定

設定公司基本資訊及打卡相關參數。

設定項目說明

欄位 說明
公司名稱 公司正式名稱（必填）
地址 公司地址
GPS 緯度 / 經度 公司所在的 GPS 座標，作為打卡圍欄的中心點
圍欄半徑（公尺） GPS 打卡圍欄的有效半徑，預設 200 公尺
Wi-Fi SSID 公司指定的 Wi-Fi 名稱，員工打卡時必須連接此 Wi-Fi 才算正常打卡
LINE Channel ID LINE Login 的 Channel ID（由 LINE Developers Console 取得）
LINE Channel Secret LINE Login 的 Channel Secret（留空表示不更改）
人事部門 指定人事部門，系統會用於特定流程的判斷

操作步驟

  1. 在左側選單點擊「基礎資料 > 公司設定」
  2. 修改需要變更的欄位
  3. 點擊「儲存設定」按鈕
公司設定 — 設定公司基本資訊、GPS 座標、Wi-Fi SSID 等
⚠️ 重要提醒 修改 Wi-Fi SSID 或 GPS 座標會直接影響員工的打卡驗證。請確認設定正確後再儲存。若公司未設定 Wi-Fi SSID，系統會退回純 GPS 模式驗證。

部門管理

管理公司的組織架構，建立、編輯部門並指定部門主管。

檢視部門列表

  • 頁面顯示所有部門：部門名稱、部門主管、狀態

新增部門

  1. 點擊「＋ 新增部門」按鈕
  2. 輸入部門名稱
  3. 選擇部門主管（從員工清單中選擇）
  4. 點擊「確定」儲存
部門管理 — 部門列表
新增部門 — 輸入名稱並指定主管

編輯 / 刪除部門

  • 點擊「編輯」可修改部門名稱和主管
  • 點擊「刪除」可移除部門；若部門下仍有員工，無法刪除，系統會提示「有 X 位員工在此部門，無法刪除」

員工管理

管理全公司員工資料，包含新增、編輯、停用員工，以及 LINE 綁定和代理人設定。

檢視員工列表

  • 可使用搜尋框以姓名或工號搜尋
  • 可依部門和狀態（在職 / 停用）篩選
  • 表格欄位：員工編號、姓名、部門、職稱、到職日、角色、LINE 綁定狀態、在職狀態
  • 操作：編輯、代理人設定、LINE 綁定碼、停用/啟用
員工管理 — 員工列表（可搜尋、篩選部門與狀態）

打卡驗證機制

員工透過 APP 打卡時，系統採用 Wi-Fi + GPS 雙重驗證：

場景 Wi-Fi GPS 結果
正常辦公 已連接公司 Wi-Fi 圍欄內 ✅ 正常打卡
Wi-Fi 正常但 GPS 偏移 已連接公司 Wi-Fi 圍欄外 ✅ 正常打卡（Wi-Fi 優先）
GPS 在範圍但無 Wi-Fi 未連接 圍欄內 ❌ 範圍外 → 需外勤打卡
Wi-Fi 未開啟 未開啟 任意 ⚠️ 提醒開啟，否則外勤打卡
外勤 任意 任意 外勤打卡（需附照片）
💡 為何需要 Wi-Fi 驗證？ GPS 圍欄通常設定 100～200 公尺，員工在公司樓下或隔壁建築也可能通過 GPS 驗證。Wi-Fi 驗證可確認員工手機確實連接到公司內部網路，大幅提高打卡的真實性。

打卡

打卡頁面是 APP 的首頁，進入後系統會自動偵測目前的位置與 Wi-Fi 狀態，判斷是否在公司範圍內。

打卡驗證方式

系統同時採用 GPS 定位與Wi-Fi SSID 雙重驗證：

  • GPS 驗證：根據手機定位座標，計算與公司位置的距離，在設定的圍欄半徑範圍內視為「在公司」
  • Wi-Fi 驗證：偵測目前連線的 Wi-Fi 名稱，與公司設定的 SSID 比對。若公司啟用「Wi-Fi 必要模式」，則必須連上公司 Wi-Fi 才能正常打卡

  5. 打卡成功後頁面會更新顯示打卡時間
  6. 下班時按下「下班打卡」按鈕，流程相同

  • 打卡按鈕有脈搏動畫效果，幫助快速辨識
  • 若 Wi-Fi 必要模式啟用但手機未開啟 Wi-Fi，系統會提示開啟 Wi-Fi 設定
  • 若 GPS 權限被拒絕，系統會引導開啟定位權限
  • 上班打卡超過班別遲到標準即記為「遲到」；下班打卡早於班別結束時間即記為「早退」
打卡頁面 — 已完成上下班打卡，顯示兩次打卡時間

申請單

點擊底部「申請單」頁籤，進入申請單管理頁面。頂部有三個子頁籤：請假、加班、補卡，可切換檢視不同類型的申請。

申請單列表頁 — 顯示三個子頁籤（請假 / 加班 / 補卡）與申請紀錄列表

請假申請

  1. 在「請假」子頁籤下，點擊右下角的「+」浮動按鈕
  2. 進入請假申請表單：
    • 選擇假別（特休、事假、病假等，僅顯示有額度的假別）
    • 選擇的假別下方會顯示可用餘額
    • 選擇起始日期 / 時間和結束日期 / 時間
    • 系統自動計算請假時數
    • 填寫請假事由
  3. 確認無誤後點擊「送出」按鈕
  4. 申請送出後狀態為「待審核」，等待主管簽核
請假申請表單 — 假別選擇、日期時間選擇、餘額顯示、事由欄位
請假申請詳情 — 顯示已送出申請的完整資訊與審核狀態

加班申請

  1. 切換到「加班」子頁籤，點擊「+」浮動按鈕
  2. 填寫加班申請表單：
    • 選擇加班日期
    • 選擇起始時間和結束時間
    • 系統自動計算加班時數
    • 填寫加班事由
  3. 確認後點擊「送出」
加班申請表單 — 日期、時間區間、事由欄位

補卡申請

  1. 切換到「補卡」子頁籤，點擊「+」浮動按鈕
  2. 填寫補卡申請表單：
    • 選擇補卡日期（限事發後 3 個日曆天內）
    • 選擇補卡時間
    • 選擇補卡類型（上班 / 下班）
    • 填寫補卡原因
  3. 確認後點擊「送出」
補卡申請表單 — 日期、時間、類型選擇、原因欄位
⚠️ 補卡限制
  • 僅能補卡3 個日曆天內的紀錄
  • 同一天最多2 筆補卡申請
  • 已有 2 筆打卡紀錄的日期不可再補卡

申請狀態查看

每筆申請會以卡片方式呈現，顯示申請類型、日期、狀態標籤：

  • 待審核 — 等待主管處理
  • 已核准 — 主管已批准
  • 已駁回 — 主管已退回，可查看駁回原因
  • 已撤銷 — 員工自行撤銷

點擊任一申請卡片可進入詳情頁面，查看完整申請資訊與簽核結果。

出勤紀錄

點擊底部「出勤紀錄」頁籤，以月曆檢視方式查看整月的出勤狀況。

月曆畫面

  • 每個日期格子會以顏色標記標示當日出勤狀態（正常、異常、請假、缺勤等）
  • 點擊任一日期可快速展開該日的打卡紀錄詳情：上班時間、下班時間、打卡方式、是否異常
  • 如該日有異常（遲到、早退、缺勤），可直接從月曆點擊快速補卡
  • 使用左右箭頭或滑動切換月份
出勤紀錄月曆 — 整月出勤狀態總覽，以顏色標記正常 / 異常 / 請假 / 缺勤
出勤紀錄日詳情 — 點擊日期後展開的打卡時間與狀態資訊

月報表

在出勤紀錄頁面右上角點擊「月報表」按鈕，進入個人月報表畫面。

報表內容

  • 統計磚塊：出勤天數、遲到次數、早退次數、請假天數、加班時數等快速總覽
  • 每日紀錄清單：依日期排列的詳細打卡紀錄，包含上下班時間、打卡方式、工時與異常標記
  • 可切換月份查看不同月份的統計資料
月報表統計磚塊 — 出勤天數、遲到 / 早退次數、請假天數等統計數字
月報表每日紀錄 — 依日期排列的打卡紀錄詳情

通知中心

點擊底部「通知」頁籤，查看所有系統通知。底部頁籤的通知圖示會顯示未讀數量紅點。

通知類型

  • 簽核結果：申請被核准或駁回時收到通知
  • 異常提醒：打卡異常（遲到、早退、缺勤）的提醒
  • 系統公告：公司管理員發送的公告訊息

操作方式

  • 未讀通知以醒目樣式顯示，已讀則為較淡的樣式
  • 點擊通知可標示為已讀並檢視詳細內容
  • 簽核結果類通知點擊後可直接跳轉到對應的申請詳情頁面
通知列表 — 顯示已讀與未讀通知，含簽核結果、異常提醒等
通知頁籤紅點 — 底部導覽列通知圖示上的未讀數量提示

頁面內容

個人資料頁面 — 個人資訊卡片、LINE 綁定狀態、快捷功能選項
變更密碼畫面 — 輸入目前密碼與新密碼

附錄

常見問題（FAQ）

Q：員工忘記密碼怎麼辦？

請聯繫管理員，由管理員在「員工管理」中重設該員工的密碼。

Q：員工離職後如何處理？

管理員在「員工管理」中將該員工帳號停用即可。停用後員工無法登入，但所有歷史出勤與申請紀錄皆保留不受影響。

Q：為什麼員工無法申請某假別？

可能原因：

  • 該假別在「假別設定」中被設為停用
  • 該假別屬於「後端給假」類型（如婚假、喪假），管理員尚未在「假期額度」中為員工建立額度
  • 該假別的年度額度已用完

Q：補卡申請為什麼無法送出？

補卡有以下限制，不符合則無法申請：

  • 已超過事發後 3 個日曆天
  • 該日已有 2 筆以上打卡紀錄
  • 該日已有 2 筆補卡申請

Q：特休天數什麼時候更新？

管理員在每年初至「假期額度」頁面執行「初始化特休額度」，系統會依據年資規則與員工到職日自動計算。

Q：如何匯出報表？

在「月出勤報表」或「薪資輔助報表」頁面，點擊右上角的「匯出 Excel」按鈕即可下載。
